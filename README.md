## Installation 安裝專案

1.從 github 下載 Reastaurant List 資料夾

```
$ git clone https://github.com/lingnli/A1RestaurantList
```

2.開啟 Terminal，進入此專案資料夾

```
cd A1RestaurantList
```

3.使用 npm 安裝套件

```
$ npm install
```

3.執行專案

```
$ npm run dev
```

並開啟網址
`http://localhost:3000`

## Features 專案功能

- 首頁所有餐廳
- 點擊特定餐廳會顯示詳細資料，包含電話、地址及餐廳描述
- 搜尋功能，在 search bar 中輸入餐廳名稱即可找到指定餐廳
